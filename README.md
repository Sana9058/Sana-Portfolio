## ðŸŒ Setup and Installation  

1. **Clone the repository:**  
    ```bash
    git clone https://github.com/Sana9058/Sana-Portfolio.git
    ```

2. **Navigate to the project directory:**  
    ```bash
    cd Sana-Portfolio
    ```

3. **Local Development:**  
    * Open the `index.html` file directly in your browser.  
    * The portfolio will run locally without needing any server.  

4. **Firebase Hosting (Optional):**  
    * Install Firebase CLI:  
      ```bash
      npm install -g firebase-tools
      ```  
    * Login to Firebase:  
      ```bash
      firebase login
      ```  
    * Initialize Firebase project:  
      ```bash
      firebase init
      ```  
    * Deploy your portfolio:  
      ```bash
      firebase deploy
      ```
